What Are Carrier Furnace Codes 13 And 33?

Carrier furnace code 13 is a limit circuit lockout error, which means that the furnace has tripped its limit switch and will not reset until it has cooled down. This can be caused by a variety of issues, such as a clogged air filter, a blocked vent or flue, or a malfunctioning blower motor.

Carrier furnace code 33 is a limit circuit fault error, which means that the furnace is detecting a problem with the limit circuit. This can be caused by a variety of issues, such as a malfunctioning limit switch, a faulty control board, or a wiring issue.

These codes are usually displayed on the control panel of the furnace, either through a series of flashing lights or through an error code display. It is important to consult the manual or wiring diagram of your specific furnace model to determine how to interpret these codes and what actions to take.

Causes Of Carrier Furnace Codes 13 And 33

There are several common causes of Carrier furnace code 13 and 33.

There are several common causes of insufficient airflow through the heat exchanger, which can trigger a code 13 error on a Carrier furnace. Some of the most common causes include:

  1. Plugged air filters: When the air filter in your furnace becomes clogged with dust and debris, it can restrict airflow through the furnace and cause the heat exchanger to overheat.
  2. Blocked discharge vents: If the discharge vents that allow hot air to escape from the furnace are blocked or restricted, it can cause the furnace to overheat and trigger a code 13 error.
  3. Obstructed air ducts: If the air ducts that distribute the heated air throughout your home are blocked or damaged, it can cause the furnace to work harder to heat your home, leading to overheating.
  4. Dirty heat exchanger: A dirty heat exchanger can also restrict airflow and cause the furnace to overheat.
  5. Failed blower motor: If the blower motor, which helps circulate heated air throughout your home, is not functioning properly, it can cause the furnace to overheat and trigger a code 13 error.
  6. Malfunctioning Blower: The blower in the furnace is responsible for moving air through the unit and out into the ductwork. If the blower is not functioning properly, it can cause the furnace to overheat and trigger the limit circuit.
  7. Malfunctioning Limit Switch: The limit switch is a safety feature that is designed to shut off the furnace if it overheats. If the limit switch is not functioning properly, it can cause the furnace to display code 13 and shut down.

It’s important to regularly maintain your furnace by checking and replacing the air filter, cleaning the heat exchanger, and inspecting the air ducts to prevent these issues from occurring. If you’re experiencing a code 13 error and are unsure of the cause, it’s best to call a professional HVAC technician for help.

Some of the possible causes of code 33 (limit circuit fault) include:

  • Malfunctioning limit switch: The limit switch is a safety device that is designed to shut off the furnace if it gets too hot. If the limit switch is malfunctioning, it can cause code 33 to be displayed.
  • Faulty control board: The control board is the “brain” of the furnace, and it is responsible for controlling all of the furnace’s functions. If the control board is faulty, it can cause code 33 to be displayed.
  • Wiring issue: If there is a problem with the wiring in the furnace, it can cause code 33 to be displayed. This could include loose or damaged wiring or incorrect wiring connections.

It is important to note that these are just some of the possible causes of codes 13 and 33. There could be other issues that may cause these codes to be displayed, so it is important to consult the manual or wiring diagram of your specific furnace model to determine the specific cause of the problem.

How To Troubleshoot Carrier Furnace Code 13 And 33

If your Carrier furnace is displaying code 13 or 33, there are several steps you can take to troubleshoot the issue. Here are some general guidelines to follow:

Consult The Manual Or Wiring Diagram Of Your Furnace

The first step in troubleshooting any furnace issue is to consult the manual or wiring diagram of your specific furnace model. This will help you understand what the codes mean and what actions to take.

Check The Air Filter

One of the most common causes of code 13 (limit circuit lockout) is a dirty or clogged air filter. Check the air filter and see if it needs to be cleaned or replaced.

Check The Vent And Flue

Make sure that the vent and flue that carry exhaust gases out of the home are not blocked or restricted. This can cause the furnace to overheat and trip the limit switch.

Check The Blower Motor

If the blower motor is not functioning properly, it can cause the furnace to overheat and trip the limit switch. Check to make sure that the blower motor is working properly.

Check The Limit Switch

If the limit switch is malfunctioning, it can cause code 33 (limit circuit fault) to be displayed. Check the limit switch and see if it needs to be replaced.

Check The Control Board

The control board is the “brain” of the furnace, and it is responsible for controlling all of the furnace’s functions. If the control board is faulty, it can cause code 33 to be displayed. Check the control board and see if it needs to be replaced.

Check The Wiring

If there is a problem with the wiring in the furnace, it can cause code 33 to be displayed. Check for loose or damaged wiring, and make sure that all of the wiring connections are correct.

If you are unable to resolve the issue on your own, it is recommended that you contact a professional HVAC technician for further assistance. A technician will be able to diagnose the problem and recommend the appropriate repairs.

Check The Furnace’s Slope

Make sure that the furnace is installed on a slope of about 1/8th of an inch from the back to the front of the unit. If the slope is not correct, it can cause condensate to build up in the secondary heat exchanger and restrict airflow, leading to a code 13 error.

Check The Furnace’s Ignition System

If the furnace’s ignition system is not functioning properly, it can cause the furnace to overheat and trigger a code 13 error. Have a professional HVAC technician inspect the ignition system and repair or replace it if necessary.

Check For Gas Leaks

A gas leak in the furnace can cause the furnace to overheat and trigger a code 13 error. If you suspect a gas leak, immediately turn off the gas supply to the furnace and call a professional HVAC technician for help.

How To Fix Carrier Furnace Code 13

If your Carrier furnace is displaying code 13, or limit circuit lockout, there are a few steps you can take to try and fix the problem:

Clean Or Replace The Air Filters

One of the most common causes of code 13 is a blocked air filter or vent. If your air filters are dirty or clogged, it can restrict the airflow through the furnace and cause it to overheat. To fix this issue, you can try cleaning the air filters or replacing them with new ones.

Clear Any Blockages From The Vents

If the air vents in your furnace are blocked or clogged, it can also restrict the airflow and cause the furnace to overheat. To fix this issue, you can try clearing any blockages from the vents using a vacuum or brush.

Replace The Limit Switch

If the limit switch is not functioning properly, it can cause the furnace to display code 13 and shut down. In this case, you may need to replace the limit switch in order to fix the problem.

If you are not comfortable attempting these repairs on your own, it is recommended that you call a professional HVAC technician to diagnose and repair your furnace. They will be able to identify the specific cause of the problem and make the necessary repairs to get your furnace up and running again.

Resetting The Carrier Furnace

If you have encountered error code 13 on your Carrier furnace, there are a few steps you can take to reset the system and try to resolve the issue. Here is how to reset your Carrier furnace:

  1. Turn off the power to the furnace. This can typically be done by flipping the switch on the electrical panel that controls the furnace. Alternatively, you can unplug the furnace or turn off the circuit breaker that supplies power to it.
  2. Once the power is off, wait a few minutes before turning it back on. This allows the furnace to reset and clear any errors that may be causing the issue.
  3. Turn the power back on by flipping the switch on the electrical panel, plugging the furnace back in, or turning the circuit breaker back on.
  4. Once the power is restored, the furnace will go through its normal start-up procedure. This may take a few minutes, during which time you may hear the furnace start up and then shut off again.
  5. If the furnace is able to start up successfully, the error code should be cleared and the furnace should return to normal operation.
  6. If the error code persists, or if the furnace is unable to start up at all, there may be a more serious issue that requires the attention of a professional HVAC technician.

More Tips

Here are a few additional tips for troubleshooting Carrier fault code 13 or any other furnace issues:

Check The Thermostat

If your furnace is not turning on or is not operating properly, it is possible that the problem could be with the thermostat. Check the thermostat to ensure that it is set to the correct temperature and that it is functioning properly.

Check The Furnace’s Electrical Connections

If you have checked the thermostat and all other visible components and the furnace is still not functioning properly, it is possible that there may be an issue with the electrical connections. Check the furnace’s wiring and connections to make sure that everything is secure and functioning properly.

Check The Furnace’s Fuel Source

If your furnace is not turning on or is not operating properly, it is possible that there could be an issue with the fuel source. If your furnace uses natural gas, check the gas valve and make sure that it is open. If your furnace uses oil, make sure that you have enough fuel in the tank.

Check The Furnace’s Safety Features

Many furnaces are equipped with safety features that will shut off the furnace if certain conditions are met. These features can include a high limit switch, a flame sensor, or a pressure switch. If any of these features are tripped, it could cause the furnace to shut down or malfunction. Check these features to make sure that they are functioning properly.

Follow The Manufacturer’s Instructions

If you are having trouble troubleshooting your furnace, it is always a good idea to consult the manufacturer’s instructions. These instructions should provide detailed information on how to troubleshoot and resolve common issues with your furnace.
